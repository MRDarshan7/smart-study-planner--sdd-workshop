# 📚 Smart Study Planner

A clean and lightweight productivity web app for students to organize study tasks without clutter.

This branch contains the **base implementation** of the Smart Study Planner before intelligent prioritization features were added.

## Features

- Add study tasks
- Subject tagging
- Deadline selection
- Difficulty tracking
- Priority selection
- Delete tasks
- LocalStorage persistence
- Responsive and clean UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- LocalStorage

## Purpose of this Branch

This branch acts as the **baseline application** for comparing two development approaches:

1. **Vibe Coding (raw prompting)**
2. **Spec-Driven Development (SDD) using OpenSpec**

No smart prioritization logic exists in this branch.
